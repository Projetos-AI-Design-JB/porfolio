# Design Specification: LEGENDARY

## 1. Aesthetic Identity (Cinematic Thinking Level 1)
- **Vibe:** Elite Technological / Modern Alchemy.
- **Atmosphere:** Clean, expensive, and mysterious. 
- **Core Principle:** "Less is More, but better."

## 2. Visual Foundation
### Color Palette
- **Obsidian Black (#000000):** Deep background for premium contrast.
- **Cyber Blue (#06B6D4):** Cyan/Blue accents for borders, buttons, and keywords.
- **Silk White (#FFFFFF):** Primary text for maximum legibility.
- **Slate Gray (#1E1E1E):** Subtle card backgrounds and dividers.

### Typography
- **Primary (Sans):** Inter or Outfit (Modern, geometric, clean).
- **Secondary (Serif Italic):** Playfair Display or EB Garamond (For sophisticated manifesto quotes).

## 3. Key UI Components
- **The "Pill" Navigation:** Floating blur-glass pill for the portfolio back-button.
- **Manifesto Section:** Transition from a dark background to a high-contrast white background with centered serif typography (Playfair Display).
- **Program Cards:** Large, rounded cards with hover-glow effects and cyan highlights.
- **Section IDs:** manifesto, chapters, legion (for functional navigation).

## 4. Interaction Design
- **Reveal on Scroll:** Elements fade-in and scale-up using GSAP.
- **Custom Cursor:** Subtle dot cursor with expansion on interactive elements.
- **Micro-animations:** Glow effects following the mouse position on cards.

## 5. Mobile Strategy
- Typography scaling (3xl for titles, 14px for body).
- Vertical stack for program cards.
- Sticky CTA button for high conversion.
